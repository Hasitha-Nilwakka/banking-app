Premium Bank - Web site, customer portal and internal system

What is this project about
This project is based on a hypothetical bank called Premium bank. The bank operates as one main branch for all customers
The project includes fully functioning web site of the bank
A customer online portal where they can manage loans
A employee portal with internal banking system 
A system admin panel to handle administrative tasks

The stack
Frontend of the applications are built with ReactJS with TypeScript
Backend is build with TypeScript and Express
Database is PostgreSQL

Applications
web - public website of the bank
customer - customer portal for account management
employee - Employee portal
admin - System Admin portal
